### How to install

1. Git clone this repository
2. Open Typhora
3. Go to TYPHORA > SETTINGS
4. Go to ASPECT > OPEN THEME DIRECTORY
5. Copy and paste the 'acid' folder and acid.css file
6. Re-start Typhora
7. Pick **Acid** from THEME menu on the top
8. Enjoy!
